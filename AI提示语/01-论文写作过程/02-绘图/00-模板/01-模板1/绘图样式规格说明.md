
## 画布尺寸

| 参数 | 值 |
|------|-----|
| 图形尺寸 | **(3.5, 2.3)** 英寸（宽 × 高） |
| 显示 DPI | **150** |
| 保存 DPI | **300** |
| 保存边距模式 | `bbox_inches='tight'` |
| 保存内边距 | `pad_inches=0.05` |
| 输出格式 | PNG + PDF |

## 字体

| 层级 | 字体 | 字号 |
|------|------|------|
| 全局 | Times New Roman, serif | 11 |
| 轴标签 | Times New Roman (x) / 宋体 SimSun (中文) | 10 |
| 轴标题 | Times New Roman / 宋体 | 10 |
| 刻度标签 | Times New Roman | 10 |
| 图例 | 宋体 SimSun (通过 FontProperties) | 9 |
| 数学文本 | STIX (`mathtext.fontset`) | — |

- 中文字体文件: `C:/Windows/Fonts/simsun.ttc`
- 负号禁用 Unicode 负号 (`axes.unicode_minus: False`)

## 坐标轴

| 属性 | 值 |
|------|-----|
| 轴线宽 | **1.0** |
| 刻度方向 | **朝内** (`direction: 'in'`) |
| 主刻度宽度 | **1.0** |
| 主刻度长度 | **5** |
| 次刻度 | 无（上/右边框刻度隐藏） |
| 上/右边框标签 | 隐藏（`labeltop: False, labelright: False`） |

## 边框 (Spines)

- 四边全部保留
- 线宽: **1.0**
- 颜色: **黑色** `'0.0'`

## 网格

| 属性 | 值 |
|------|-----|
| 显示 | 是 (`ax.grid(True)`) |
| 对应刻度 | 仅主刻度 (`which='major'`) |
| 线型 | 虚线 (`'--'`) |
| 线宽 | **0.4** |
| 透明度 | **0.5** |

## 曲线颜色

| 元素 | 颜色 | 透明度 | 线宽 |
|------|------|--------|------|
| 填充区域 (最大–最小范围) | `#acd9fd` (浅蓝) | 0.6 | 0 (无边线) |
| 均值曲线 | `#37a0f3` (中蓝) | 1.0 | **1.0** |
| 均值线型 | 实线 (`'-'`) | — | — |

## 图例

| 属性 | 值 |
|------|-----|
| 位置 | `'lower right'` |
| 锚点偏移 | `bbox_to_anchor=(0.98, 0.05)` |
| 图例句柄长度 | **1.5** |
| 句柄与文本间距 | **0.4** |
| 边框内边距 | **0.3** |
| 显示边框 | 是 |
| 边框颜色 | `'0.8'` (浅灰, 约 #cccccc) |
| 圆角边框 | 否 (`fancybox: False`) |

## 横轴格式

- 原始步数除以 **10⁷** 显示
- 无小数 (`:.0f`)
- 使用 `FuncFormatter` 自定义
- 标签: `训练步数（×10⁷）`

## 布局

- `plt.tight_layout()` 自动调整

## 复现代码模板（核心 rcParams）

```python
import matplotlib.pyplot as plt
import matplotlib as mpl
from matplotlib.font_manager import FontProperties

zh_font = FontProperties(fname='C:/Windows/Fonts/simsun.ttc', size=10)

plt.rcParams.update({
    'font.family': 'serif',
    'font.serif': ['Times New Roman'],
    'axes.unicode_minus': False,
    'font.size': 11,
    'axes.linewidth': 1.0,
    'axes.labelsize': 10,
    'axes.titlesize': 10,
    'xtick.labelsize': 10,
    'ytick.labelsize': 10,
    'xtick.major.width': 1.0,
    'ytick.major.width': 1.0,
    'xtick.direction': 'in',
    'ytick.direction': 'in',
    'xtick.major.size': 5,
    'ytick.major.size': 5,
    'legend.fontsize': 9,
    'legend.frameon': True,
    'legend.edgecolor': '0.8',
    'legend.fancybox': False,
    'figure.dpi': 150,
    'savefig.dpi': 300,
    'savefig.bbox': 'tight',
    'savefig.pad_inches': 0.05,
    'mathtext.fontset': 'stix',
})

fig, ax = plt.subplots(figsize=(3.5, 2.3))

# 填充 + 曲线
ax.fill_between(x, y_lower, y_upper, alpha=0.6, color='#acd9fd', linewidth=0)
ax.plot(x, y_mid, color='#37a0f3', linewidth=1.0, linestyle='-')

# 刻度
ax.tick_params(which='both', top=False, right=False, labeltop=False, labelright=False)
ax.tick_params(axis='both', which='major', direction='in')

# 网格
ax.grid(True, which='major', linestyle='--', linewidth=0.4, alpha=0.5)

# 边框
for spine in ax.spines.values():
    spine.set_linewidth(1.0)
    spine.set_color('0.0')

# 图例
ax.legend(loc='lower right', bbox_to_anchor=(0.98, 0.05),
          handlelength=1.5, handletextpad=0.4, borderpad=0.3, prop=zh_font)

plt.tight_layout()
plt.savefig('output.png')
plt.savefig('output.pdf')
```

# 绘图样式规格说明

## 画布尺寸

| 参数 | 值 |
|------|-----|
| 图形尺寸 | **(3.5, 2.3)** 英寸（宽 × 高） |
| 显示 DPI | **150** |
| 保存 DPI | **300** |
| 保存边距模式 | `bbox_inches='tight'` |
| 保存内边距 | `pad_inches=0.05` |
| 输出格式 | PNG + PDF |

## 字体

| 层级 | 字体 | 字号 |
|------|------|------|
| 全局 | Times New Roman, serif | 11 |
| 轴标签 | Times New Roman (x) / 宋体 SimSun (中文) | 11 |
| 轴标题 | Times New Roman / 宋体 | 11 |
| 刻度标签 | Times New Roman | 11 |
| 图例 | 宋体 SimSun (通过 FontProperties) | 11 |
| 数学文本 | STIX (`mathtext.fontset`) | — |
  

- 中文字体文件: `C:/Windows/Fonts/simsun.ttc`
- 负号禁用 Unicode 负号 (`axes.unicode_minus: False`)

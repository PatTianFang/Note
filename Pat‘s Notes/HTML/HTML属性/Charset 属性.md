HTML <meta> charset 属性

E.g.
```HTML
<html>
	<head>
		<meta charset="UTF-8">
	</head>
	
	<body>
		<h1>My Website</h1>
		<p>Some text...</p>
	</body>
</html>
```

提示：charset 属性可以通过任意元素上的[[HTML lang 属性|Lang属性]]来重写。

规定 HTML 文档的字符编码：
<head>
<meta charset="UTF-8">
</head>

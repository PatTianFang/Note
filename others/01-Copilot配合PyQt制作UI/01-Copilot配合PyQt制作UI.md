# Copilot配合PyQt制作UI

主要思路是使用`Python`写逻辑，`PyQt`写界面UI，最后通过`Copilot`对UI进行美化。

关键的提示词是**WindowsFluent**设计风格

先使用`conda`创建虚拟环境`UI`

![Alt](Pasted%20image%2020250411191725.png)

```conda
conda create -n UI python=3.11
```

激活环境

```conda
conda activate UI
```

然后下载第三方库`PyQt`

![Alt](Pasted%20image%2020250411191858.png)

```pip
pip install PyQt5
```

在`Copilot`中输入你想要的关键词，比如
`使用Pyqt5和WindowsFluent设计风格制作一个2048游戏`

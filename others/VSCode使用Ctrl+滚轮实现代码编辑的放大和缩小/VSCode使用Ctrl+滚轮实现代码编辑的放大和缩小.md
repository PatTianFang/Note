# VSCode使用Ctrl+滚轮实现代码编辑的放大和缩小


文章主要参考CSDN从入门到躺下的文章《VScode使用Ctrl+滚轮实现代码编辑的放大和缩小》^[https://blog.csdn.net/qq_45078197]

打开VSCode，点击左下角的设置。

![Alt](file-20250505092053081.png)

点击设置

![Alt](file-20250505092131571.png)

在搜素框输入“zoom”

![Alt](file-20250505092519695.png)

勾上文本编辑选项

![Alt](file-20250505092552477.png)

即可实现代码编辑的放大和缩小

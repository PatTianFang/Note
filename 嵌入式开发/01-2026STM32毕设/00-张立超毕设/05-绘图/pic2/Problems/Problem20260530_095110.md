## 任务目标

在 `pic2` 中生成一张黑白线条电路图，要求无标题、无背景色、线条清晰适合 A4 打印。图中需要以 VHF 对讲核心子板 U2 为中心，展示电源去耦、控制引脚上拉、外部接口、RF 输入输出，以及子板内部功能模块与信号流向。

## 思考过程

本任务是可调节的工程绘图，优先使用代码生成 SVG 矢量图，而不是位图。SVG 能保持打印清晰度，也方便后续调整页面尺寸、线宽、字体和布局坐标。

布局假设：U2 位于画面中心，左侧放电源、控制、麦克风和 RF 输入，右侧放 LED、功放、串口和 RF 输出。子板内部用虚线框表示四个功能模块，并用箭头标出 MIC_IN 到 RF_OUT、RF_IN 到 AF_OUT 两条信号流。

## 遇到的问题

潜在问题是功能模块名称中同时要求“FM调制解调”和箭头中的“FM调制 / FM解调”，为保持清晰，图中放置一个“FM调制解调”虚线框，并在框内标注“FM调制”和“FM解调”两条路径。

生成和预览过程中遇到的问题：

- 初版左侧电源、C7/C8、R5/R6 上拉电阻布局过密，标注有重叠。
- TXLED/RXLED 外部 LED 行距不足。
- `reportlab` 渲染 SVG 预览时缺少中文字体，中文显示为方框。
- 内部 SQ 箭头初版为斜线，容易和 RF_IN 接收路径产生视觉交叉。

## 解决方法

创建 `pic2/draw_pic2.py`，使用 Python 标准库输出透明背景 SVG。图中所有对外引脚使用实线连接，接地符号使用统一函数绘制，内部模块使用虚线框和箭头。

修正过程：

- 将 U2 整体右移，给左侧电源和上拉网络留出独立空间。
- 将 C7/C8 标注分到两侧，减少文字重叠。
- 将 LED1/LED2 垂直拉开，TXLED/RXLED 连接保持水平。
- 将 SQ 内部箭头改为折线路径，避免与 RF_IN 路径交叉。
- 使用 Edge 无头模式从 SVG 导出 PNG/PDF，解决普通渲染器中文字体缺失的问题。

## 验证结果

已完成验证：

- 运行 `python pic2\draw_pic2.py` 成功生成 `pic2/pic2_circuit.svg`。
- 使用 Python `xml.etree.ElementTree` 成功解析 SVG。
- 检查关键标注未缺失：U2、集成化中频射频子板、VBAT、GND、PTT、PD、TXLED、RXLED、MIC_IN、AF_OUT、SQ、TXD、RXD、RF_IN、RF_OUT、C7、C8、R5、R6、LED1、LED2、FM调制解调、本振/混频、中频放大、静噪控制。
- 使用 `svglib/reportlab` 和 Edge 无头模式分别渲染预览；Edge 导出的 PNG/PDF 能正常显示中文。

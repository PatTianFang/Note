## 任务目标

在 `pic3` 中生成一张黑白线条电路图，要求无标题、无背景色、线条清晰适合 A4 打印。图中需要绘制从核心子板 RF_OUT 到 SMA 天线座 J1、ANT1 的两级 π 型射频匹配网络，并突出射频信号线为粗实线。

## 思考过程

该任务是可调节的符号电路图，选择用 Python 生成 SVG 矢量图。SVG 适合 A4 打印，且可以在脚本中统一调整页面尺寸、线宽、字体、元件间距和符号比例。

布局假设：射频主链路从左到右水平展开，RF_OUT、L2、C4、L3、C7、R8、功率焊盘、J1、ANT1 依次排列；C4、C7 和 J1 外壳接同一系统地。0.5W 与 2W 焊盘之间使用虚线连接，仅表示可切换关系，不表示固定实线连接。

## 遇到的问题

潜在问题是 0.5W 与 2W 两个功率焊盘的实际切换方式未指定，因此图中只按要求绘制虚线切换提示，不额外添加开关或跳线器型号。

生成和预览过程中发现的问题：

- `reportlab` 渲染预览时中文字体显示为方框，因此不能作为正式预览图。
- 初版 P1/P2 焊盘的 `2W` 与 `P1` 标注距离偏近。

## 解决方法

创建 `pic3/draw_pic3.py`，使用标准库输出 SVG。电感使用螺旋符号，电容使用两条平行线，电阻使用矩形，天线使用标准天线符号，射频主线使用较粗实线。

修正过程：

- 将 P2 的 2W 焊盘上移，使 P1/P2 标注分离。
- 使用 Edge 无头模式从 SVG 导出 PNG/PDF，解决普通渲染器中文字体缺失的问题。
- 保留 `pic3_circuit.svg` 作为主文件，确保背景透明且可继续通过代码调整。

## 验证结果

已完成验证：

- 运行 `python pic3\draw_pic3.py` 成功生成 `pic3/pic3_circuit.svg`。
- 使用 Python `xml.etree.ElementTree` 成功解析 SVG。
- 检查关键标注未缺失：RF_OUT、J1、SMA、ANT1、L2、L3、C4、C7、R8、P1、P2、0.5W、2W、27nH、12pF、33Ω、2dBi、系统地。
- 使用 Edge 无头模式导出 `pic3_circuit.png` 和 `pic3_circuit.pdf`，中文显示正常。

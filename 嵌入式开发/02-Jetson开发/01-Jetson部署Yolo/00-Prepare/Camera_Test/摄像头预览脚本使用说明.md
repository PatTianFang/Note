# 摄像头预览脚本使用说明

本文说明如何使用 `camera_preview.py` 通过 OpenCV 实时显示摄像头画面。

## 1. 脚本位置

脚本位于：

```text
Camera_Test/camera_preview.py
```

## 2. 运行前检查

确认已经安装 OpenCV：

```bash
python3 -c "import cv2; print(cv2.__version__)"
```

确认摄像头设备存在：

```bash
ls /dev/video*
```

如果能看到 `/dev/video0`，通常可以直接运行脚本。

## 3. 运行脚本

进入脚本目录：

```bash
cd /home/gwb/Fang/Camera_Test
```

直接运行：

```bash
python3 camera_preview.py
```

脚本不需要传入任何参数，默认打开摄像头编号 `0`，通常对应 `/dev/video0`。

## 4. 退出窗口

摄像头窗口打开后，点击窗口使其获得焦点，然后按下面任意一个按键退出：

```text
q
Esc
```

## 5. 常见问题

如果提示无法打开摄像头，检查摄像头是否连接：

```bash
ls /dev/video*
```

如果摄像头被其他程序占用，请先关闭其他正在使用摄像头的软件。

如果没有图形界面或没有连接显示器，OpenCV 窗口无法正常弹出，需要在桌面环境中运行。

如果系统没有安装 OpenCV，可以先安装：

```bash
sudo apt update
sudo apt install python3-opencv
```
